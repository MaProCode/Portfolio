Here below are described some of the effects / animation implemented in this website.

Main page
> Particles Cloud effect:

- It's a popular web design animation where numerous small, often luminous or semi-transparent particles are rendered on a webpage. These particles typically float and move fluidly, often connecting with each other with thin lines to form a dynamic, web-like pattern.
- It adds a modern, futuristic, and engaging layer to a website's background. It can be particularly effective for tech companies and creative portfolios.


> Typewriter effect:

- digital animation technique that simulates the appearance of text being typed in real-time.
Pro:
- It's often used on websites to grab a user's attention, highlight key messages, or create a dynamic introduction.
- It instantly makes a page feel more interactive and alive, encouraging users to pay attention.
- It's great for storytelling or presenting a step-by-step process.
- The effect has a vintage, nostalgic feel that can give a website a unique and memorable personality.


> Dark mode:

- It's a color scheme that uses light-colored text, icons, and graphical elements on a dark background.
- It's the opposite of the traditional "light mode," which uses dark text on a light background.
Pro:
- Reduces Eye Strain: It can be more comfortable for your eyes, especially when you're using a device in a low-light environment or at night.
- Saves Battery Life: On devices with OLED or AMOLED screens (like many modern smartphones), dark mode can significantly reduce battery consumption because black pixels are turned off completely.
- Personal Preference: Many users simply prefer the modern, sleek aesthetic that a dark interface offers.


> Icon Slider banner:

- It's a common element on websites that displays a horizontal, often automatically scrolling series of logos or icons.
- It's typically used to showcase a company's partners, clients, or media mentions.
Pro:
- Saving space: It allows you to display multiple logos without cluttering the page.
- Highlighting key relationships: It quickly communicates who a company works with or is recognized by.
- Creating a professional look: A well-designed slider adds a polished and dynamic touch to a website's layout

Contact Page
> Neuromorphic box:

- It's a softened contour element, that create a tridimensional effect on the UI.
- It adds a modern, futuristic, and engaging touch to the UI, in an Apple-like style.
- Available in Light and Dark theme.


> Email service:

- Third-party services allows to send emails directly from a website's front-end code using JavaScript, without requiring any server-side logic.
- It is primarily used to handle form submissions, contact forms, and other email-based functionalities on static websites.


Portfolio page
> Card tilt effect:

- It's a dynamic visual effect that makes a user interface (UI) element appear to tilt in three-dimensional space in response to a user's mouse cursor movement.
- The angle of the rotation is directly proportional to the distance of the cursor from the center of the element, creating a smooth and realistic 3D illusion.
- a glare effect is applied on the card as its rotation angle increase.
- Enhance User Experience (UX): It provides subtle, engaging interactivity that makes a page feel more responsive and alive.


Project page
> Animated Timeline:

- It's a visual design element that presents a sequence of events, milestones, or data points in chronological order using motion and dynamic transitions.
- Narrative Flow: The animated sequence helps tell a story, guiding the user through a journey, such as a company's history, a product's development, or a project's phases.
- Improved User Experience: By breaking up large blocks of text and using interactive elements, it keeps users engaged and makes the information more memorable and scannable.


> Fade up effect:

- It's a simple yet popular animation that makes an element appear on the screen by gradually increasing its opacity from 0 (fully transparent) to 1 (fully opaque), or sliding in the content as the scroll of the page proceed.
- This creates the visual impression that the element is "fading in" or "fading up" into view.


> Zoom Overlay images:

- An image zoom overlay effect with a blurred background is a dynamic web design feature that provides an immersive, detailed view of a photo while visually separating it from the rest of the page.
- This effect is typically activated when a user clicks on a thumbnail or smaller version of an image.
- This combined effect creates a clean, focused, and professional look, offering a superior user experience for viewing photos in galleries, portfolios, and e-commerce product pages.
