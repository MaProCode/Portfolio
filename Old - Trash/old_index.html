Realization of electronic devices based on commercial HW, that enables data exchange between different communication buses or protocols.Them acts as a translator, converting data from one format or protocol to another.
A communication gateway sits between these buses and performs the following functions:

> Protocol Translation: It interprets the data from one protocol and reformats it to match the target protocol, such as serial bus / Bluetooth / Wi-Fi or do some kind of actuation.

> Data Mapping: It associates data fields from one system with the appropriate fields in the other, often converting values or units as needed.

> Filtering and Routing: It can selectively transmit data based on rules or address filters, improving efficiency and reducing unnecessary traffic.
